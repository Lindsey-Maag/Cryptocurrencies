# Cryptocurrencies

## Purpose
Create a report showing what cryptocurrencies are on the trading market and create a classification system for client.


## Process
- Preprocess Data
  - Only keep cryptocurrencies being traded
  - Eliminate text features for upcoming PCA process
  - Scale data
- Reduce Data Dimensions
  - Use PCA (Principal Component Analysis) to reduce dimensions to 3 principal components
- Clustering Cryptocurrencies
  - Create an Elbow curve to determine best value for K
  - Use K-means algorithm to make predictions of K-clusters
